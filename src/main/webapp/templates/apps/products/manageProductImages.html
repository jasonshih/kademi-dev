Please upload image files with the same name as the product code you want them attached to, with an extension. For example, to upload an image to product with SKU ABC-123 you could upload a file called "ABC-123.png".

Allowable formats:

  • Product code: a1.png (for a product with code "a1")
  • Product code, followed by a number: a1.2.png (for a product with code "a1", to add an extra image)
  • Variant code: a1-1.jpg (for a product which has a variant with code "a1-1"
  • Full: a1.size.large.png (product "a1", variant type "size", variant "large")
